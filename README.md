# TUKOPI - Warehouse Coffee & Roastery

Proyek ini terdiri dari dua bagian utama:

- **Frontend**: Dibangun menggunakan AstroJS (terletak di folder `/Frontend`)
- **Backend**: Dibangun menggunakan Node.js, Express, dan MySQL (terletak di folder `/Backend`)

## Struktur Proyek

- `/Frontend`: File website (Astro, CSS, JS)
- `/Backend`: API Server, Routes, dan Database SQL
- `/Figma`: File desain atau referensi visual

## Cara Menjalankan

### 1. Menjalankan Backend

1. Masuk ke folder backend: `cd Backend`
2. Instal dependensi: `npm install`
3. Pastikan MySQL berjalan dan import file `tukopi.sql`
4. Sesuaikan konfigurasi di file `.env`
5. Jalankan server: `npm run dev` (menggunakan nodemon)

### 2. Menjalankan Frontend

1. Masuk ke folder frontend: `cd Frontend`
2. Instal dependensi: `npm install`
3. Jalankan website: `npm run dev`
4. Buka di browser: `http://localhost:4321`
